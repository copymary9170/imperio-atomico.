# Sistema de diseno Copy Mary ERP

## Personalidad

Copy Mary ERP debe sentirse limpio, creativo, confiable y facil de usar. Es un sistema para administrar impresiones, papeleria, ventas, inventario y ganancias sin perder dinero.

## Reglas de experiencia

1. El usuario debe ver primero ventas, ganancias, pedidos e inventario critico.
2. Toda accion importante debe tener confirmacion visual.
3. Todo error debe explicar como corregirlo.
4. No se debe permitir vender por debajo del costo.
5. El estado critico siempre debe destacarse visualmente.
6. La app debe poder usarse desde telefono.
7. Los textos deben usar acentos correctos y lenguaje claro.

## Tono de voz

Claro, directo y comercial.

Ejemplos:

- Correcto: Venta calculada correctamente.
- Error: El precio no puede ser menor que el costo.
- Alerta: Revisa este material antes de aceptar mas pedidos.
